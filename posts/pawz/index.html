This post is intended to provide a simple demonstration of a Python package that relies on core functionality written in Rust.

Tools

In addition to Python and Rust, we’ll use the following tools:

  • GitHub: repository hosting
  • GitHub CLI: interact with GitHub from the command line
  • just: a command runner
  • mise: a tool for managing dev environments (Python/Rust versions)

These will be explained along the way. For generating and using Rust code in Python, we’ll use PyO3.
